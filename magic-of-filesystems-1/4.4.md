# chroot: let's change the repository

1. Install minimal rootfs structure:
```shell
mkdir -p fstest/alpine-minirootfs && cd fstest/alpine-minirootfs
curl -L https://dl-cdn.alpinelinux.org/alpine/v3.23/releases/x86_64/alpine-minirootfs-3.23.0-x86_64.tar.gz
tar xzvf alpine-minirootfs-3.23.0-x86_64.tar.gz
```
2. Bind necessary pseudo-files:
```shell
sudo mount -o bind /dev dev
sudo mount -o bind /proc proc
sudo mount -o bind /sys sys
```
3. Run chroot:
```shell
sudo chroot . /bin/sh
```
4. Try to install Alpine package:
```shell
echo "nameserver 8.8.8.8" >> /etc/resolv.conf
apk add vim
```
5. Clean:
```shell
exit
sudo umount dev
sudo umount  proc
sudo umount sys
```
