# 4.2. tmpfs: file storage in RAM

1. Mount a 2GiB tmpfs filesystem:
```shell
mkdir -p mytmpfs
sudo mount -t tmpfs -o size=2g no_device ./mytmpfs
```
2. Check memory usage:
```shell
free -ht
               total        used        free      shared  buff/cache   available
Mem:            30Gi       8.0Gi       5.5Gi       3.1Gi        17Gi        20Gi
Swap:          2.0Gi       2.0Mi       2.0Gi
Total:          32Gi       8.0Gi       7.5Gi
```
- Memory usage is not impacted by tmpfs mounting.
3. Create a 1GiB file:
```shell
fallocate -l 1GiB ./mytmpfs/tmpfile
```
4. Check memory usage:
```shell
free -ht
               total        used        free      shared  buff/cache   available
Mem:            30Gi       8.0Gi       4.3Gi       4.2Gi        18Gi        19Gi
Swap:          2.0Gi       2.0Mi       2.0Gi
Total:          32Gi       8.0Gi       6.3Gi
```
- Memory usage increases by approximately 1GiB
5. Unmount tmpfs filesystem:
```shell
sudo umount ./mytmpfs
```
